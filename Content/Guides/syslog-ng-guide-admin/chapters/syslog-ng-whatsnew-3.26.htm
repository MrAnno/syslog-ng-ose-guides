Version of includes the following main features.

Persist name assigned to Python sources and destinations
Disabling and forcing using TLSv1.3

  • The tlsv1_3 value has been added to the ssl-version() option in http() destination-based drivers.

    For more information, see HTTP destination options and elasticsearch-http destination options.

  • The no-tlsv13 value has been added to the ssl-options() options.

    For more information, see TLS options.
