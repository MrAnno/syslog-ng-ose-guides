The Azure auth header plugin

The Azure auth header plugin is a signal-slot mechanism-based module that generates authorization headers for applications that connect to Microsoft Azure.

Example configuration
Example configuration for using the Azure auth header plugin

The following example can be copy-pasted and used as a template for using the Azure auth header plugin in your configuration.

@version: 7.0
@include 'scl.conf'
log {
  source{
    example-msg-generator(num(1) freq(1) template("<34>Oct 11 22:14:15 tristram su: new msg\n") );
};
  destination{
    sentinel(
      workspace_id("")
      auth_secret("")
    );
  };
  flags(flow-control);
};
