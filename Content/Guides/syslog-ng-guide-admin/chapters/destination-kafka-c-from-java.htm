Shifting from Java implementation to C implementation

If you were using the Java implementation of the kafka() destination and want to shift to the C implementation, consider the following:

  • Download the latest stable binary release of the Apache Kafka libraries (version 0.9 or newer) from http://kafka.apache.org/downloads.html.

  • Extract the Apache Kafka libraries into a single directory. If needed, collect the various .jar files into a single directory (for example, /opt/kafka/lib/) where can access them. You must specify this directory in the configuration file.
