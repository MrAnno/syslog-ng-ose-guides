sumologic-http()

The sumologic-syslog() and sumologic-http() destinations send log messages to sumo logic, a cloud-based log management and security analytics service.

Sending data using the sumologic-http() destination
Example: Using the sumologic-http() destination

The following example sends every log from the system() source to your Loggly account.

log {
destination
{sumologic-http(collector("UNIQUE-HTTP-COLLECTOR-CODE-AS-PROVIDED-BY-sumologic") deployment("ENDPOINT") tls(peer-verify(yes) ca-dir('/etc/syslog-ng/ca.d'))); ); }
;};

To use the sumologic-http() driver, the scl.conf file must be included in your configuration:

@include "scl.conf"
